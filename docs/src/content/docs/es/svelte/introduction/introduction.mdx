### Instalación

Instala **Fluid DnD** usando tu manejador de paquetes favorito:

```bash
# con npm:
npm i fluid-dnd

# con yarn:
yarn add fluid-dnd

# con pnpm:
pnpm i fluid-dnd
```

### Uso

Importa el **composable** (`useDragAndDrop`):

export const compositionApiCode = `
<script>
  import { useDragAndDrop } from "fluid-dnd/svelte";
</script>`;
