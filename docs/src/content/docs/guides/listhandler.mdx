### List of number example

With **Vue Fluid DnD** you dont have to drag the elements across its entire surface, just pass a **handler selector** to `handlerClass` that will be used to find the handler element as following:

export const listOfNumbers = `<script setup lang="ts">
import { ref } from "vue";
import { useDragAndDrop } from "vue-fluid-dnd";
import Handler from "./icons/handler.vue";

const list = ref([1, 2, 3, 4, 5]);
const handlerClass = "handler";
const { parent } = useDragAndDrop(list, { handlerClass });

</script>`;

<Code code={listOfNumbers} lang="vue" />

### Creating the view

After, we'll create the `template` and add `handlerClass` class to the `Handler` component:

export const listOfNumbersDraggable = `

<template>
    <ul ref="parent" class="number-list">
      <li class="number" v-for="(element, index) in list" :index="index">
        <Handler class="handler" />
        {{ element }}
    </li>
    </ul>
</template>`;

export const highlightsDraggable = ["handler"];

<Code code={listOfNumbersDraggable} lang="vue" mark={highlightsDraggable} />
